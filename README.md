## Как запустить:

Установить SBЕ и Java.

На Windows необходимо установить [Hadoop и HDFS](https://towardsdatascience.com/installing-hadoop-3-2-1-single-node-cluster-on-windows-10-ac258dd48aef)

Выполнить команду:
```
sbt run
```

Запуск тестов:
```
sbt test
```
